# Garmin Nervous-System Workflow

This workflow explains how to use Garmin data as one layer of SelfTune nervous-system reflection.

Garmin metrics and subjective reports are for pattern awareness, not diagnosis.

## Inputs

Use:

1. Garmin daily summary or screenshot
2. Subjective nervous-system check-in
3. Supplement and nutrition context
4. Mood, energy, digestion, and body-signal notes
5. AI-assisted reflection prompt

## Garmin Metrics To Capture

- Sleep duration
- Sleep score
- Body Battery
- Stress level
- Resting heart rate
- HRV status, if available
- Activity or exercise
- Notable stress windows
- Recovery or low-energy windows

## How To Interpret Metrics Safely

Use Garmin data as a cue for curiosity.

- Low Body Battery can prompt review of sleep, workload, meals, caffeine, movement, and emotional load.
- High stress readings can reflect exercise, excitement, caffeine, heat, pressure, conflict, illness, or overwork.
- HRV can be reviewed as a trend, not as a single-day verdict.
- Sleep score should be paired with subjective refreshment.

Do not treat Garmin data as proof of a nervous-system state.

## Daily Flow

1. Capture the Garmin data.
2. Fill in `templates/garmin-nervous-system-log.md`.
3. Add subjective notes about energy, mood, mental clarity, tension, digestion, and felt safety.
4. Add supplement and nutrition context.
5. Ask an AI assistant for a non-medical reflection using `prompts/garmin-supplement-nervous-system-review.md`.
6. Choose one gentle observation for tomorrow.

## Weekly Flow

At the end of the week:

1. Combine seven daily logs.
2. Use `templates/weekly-nervous-system-review.md`.
3. Ask for a pattern review with `prompts/weekly-nervous-system-pattern-review.md`.
4. Save useful observations in `templates/analysis-result-log.md`.
5. Do not convert patterns into diagnoses.

## Data Stress Reminder

Wearable data can become stressful if users treat numbers as grades.

SelfTune reframes metrics as signals:

"My body may be working hard today. What would reduce load or support recovery?"
