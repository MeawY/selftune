# Practitioner Intake / 実践者向けヒアリング

This template is for therapists, coaches, wellness practitioners, or facilitators who want to support structured reflection without providing medical diagnosis or treatment through SelfTune.

このテンプレートは、SelfTune を使って診断や治療ではなく、構造化された振り返りを支援したいセラピスト、コーチ、ウェルネス実践者、ファシリテーター向けです。

## 1. Client Context / クライアントの文脈

- Preferred name / 呼び名:
- Primary reason for using SelfTune / SelfTune を使う主な理由:
- Current routines they want to observe / 観察したい現在の習慣:
- Existing support network / 既存の支援:

## 2. Scope Agreement / 範囲の合意

- SelfTune will be used for reflection and record organization.
- SelfTune will not be used for diagnosis, treatment, or emergency support.
- Medical, psychological, nutritional, or urgent concerns should be directed to qualified professionals.

- SelfTune は内省と記録整理のために使う。
- SelfTune を診断、治療、緊急対応のために使わない。
- 医療、心理、栄養、緊急性のある相談は、資格を持つ専門家につなぐ。

## 3. Observation Areas / 観察領域

Choose only areas that are relevant and comfortable.

必要で、本人が扱いやすい領域だけを選びます。

- Sleep / 睡眠
- Energy / エネルギー
- Mood / 気分
- Stress / ストレス
- Meals / 食事
- Movement / 運動
- Body signals / 身体シグナル
- Work or study load / 仕事・学習負荷
- Relationships or social context / 人間関係・社会的文脈

## 4. Boundaries and Privacy / 境界とプライバシー

- What should not be recorded? / 記録しないこと:
- Where will notes be stored? / 記録の保存場所:
- Who can see the notes? / 閲覧できる人:
- What should be removed before sharing with AI tools? / AIツール共有前に削除する情報:

## 5. Review Rhythm / レビュー頻度

- Daily check-in / 日次チェックイン:
- Weekly review / 週次レビュー:
- Monthly review / 月次レビュー:
- Practitioner review session / 実践者とのレビュー:

## 6. Professional Support Triggers / 専門家相談の目安

Discuss in advance when the practitioner should suggest outside support.

どのような場合に外部の専門家支援を提案するか、事前に確認します。

- Severe or sudden symptoms / 強い症状・急な変化
- Persistent or worsening distress / 長引く、悪化するつらさ
- Safety concerns / 安全面の懸念
- Nutrition, medication, or treatment questions / 栄養、薬、治療に関する質問
- Anything outside the practitioner's scope / 実践者の専門範囲外の内容
