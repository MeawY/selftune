# Weekly Review / 週次レビュー

Week of / 週:

This template is for non-medical reflection on patterns across several days.

このテンプレートは、数日間のパターンを非医療的に振り返るためのものです。

## 1. Overall Summary / 全体サマリー

- What stood out this week? / 今週目立ったこと:
- What felt steadier? / 安定していたこと:
- What felt more difficult? / 難しかったこと:

## 2. Repeated Signals / 繰り返し見えたシグナル

- Sleep / 睡眠:
- Energy / エネルギー:
- Mood / 気分:
- Stress / ストレス:
- Digestion or appetite / 消化・食欲:
- Body sensations / 身体感覚:

## 3. Context Review / 文脈レビュー

- Meals and timing / 食事と時間:
- Movement / 運動・移動:
- Workload or study load / 仕事・学習量:
- Social context / 人との関わり:
- Environment / 環境:
- Rest and recovery / 休息と回復:

## 4. Patterns and Questions / パターンと問い

- Possible pattern / ありそうなパターン:
- Missing information / 足りない情報:
- Question for next week / 来週確認したい問い:

## 5. Small Experiments / 小さな試行

Choose one or two small, reversible experiments.

小さく、元に戻せる試行を1つか2つ選びます。

- Experiment 1 / 試行1:
- Experiment 2 / 試行2:

## 6. Support Flag / 相談メモ

Is there anything severe, sudden, persistent, worsening, or concerning that may be worth discussing with a qualified professional?

強い症状、急な変化、長引く不調、悪化、不安な点など、専門家に相談したほうがよさそうなことはありますか。
